Bounce

An interface, containing blurring, parallax and responsive bouncing.

About Bounce

I've seen some really cool examples of imagery scaling up when you overscroll. Also, wanted to mess around with some webkit filters. Thus, this exists. The blurring is kinda like the Medium homepage but they use a blurred image over top. Which for performance might be better. Built on skeleton cause I like using it. You go ahead and use whatever responsive framework gets you up in the morning.

Dependencies:
IScroll (for mobile scrolling and calculating)

More notes

Works on Mobile! Hence the iScroll implementation. However, having a couple issues on the ipad where it seems like it is lagging. Might be a little too much going on. Also IE & Firefox, sorry, no blurring. If you don't have a touch responsive mouse, you may not see the overscroll. Get one that does. They are fun.

The rest, is just junk.
